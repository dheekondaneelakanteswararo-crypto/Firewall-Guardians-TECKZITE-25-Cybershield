# Firewall-Guardians-TECKZITE-25-Cybershield
AI-powered phishing and scam message detector that analyzes sender details, message text, and links to generate a risk score (SAFE / WARNING / DANGER) with clear explanations and recommendations. Built as a simulation-based cybersecurity project for TeckZite 2k25 PS-5.

# App Link
https://cyber-shield-8b4a802a.base44.app

# Problem Statement
Design and implement a simulation-based Cybersecurity & Digital Safety system that can detect fake profiles and phishing attempts, analyze scam messages, assign a risk score, trigger warning alerts, and prioritize security incidents for users, using AI-driven decision logic and automation.

# Key Features
- Real-time phishing and scam message detection with a simple web interface.
- Multi-factor risk scoring (sender, content patterns, links) with SAFE / WARNING / DANGER labels.
- Clear explanations for each detected risk, helping users understand why a message is dangerous.
- Simulation-based design using dummy/synthetic data, fully aligned with hackathon rules (no real money or sensitive data).
- Lightweight web app built on Base44 and deployable as a secure, browser-based cybersecurity tool.

# Tech Stack
**Frontend:** HTML5, CSS3, JavaScript (ES6) for responsive web UI.
**Backend / Logic:** Base44 workflow engine for phishing detection rules and risk scoring.
**Data:** Synthetic/dummy email and message samples for simulation-based analysis.
**Deployment:** Base44 cloud-hosted web app at `cyber-shield-8b4a802a.base44.app`.
